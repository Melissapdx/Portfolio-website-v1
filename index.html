Milkmaids

Milkmaids is a responsive breast milk delivery web app built in Python with Javascript, Flask & SQLAlchemy. Front end design built with a mobile first design with basic Bootstrap and custom CSS styles. For parents who do not have the option of breastfeeding, they can purchase milk from donors and have it delivered to their door

Tech Stack

Frontend: Javascript, jQuery, AJAX, HTML, CSS, Bootstrap

APIs: Stripe
